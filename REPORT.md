## 1. 全体構造 — 7つの大系譜

この年表は、1890年代から2000年代にかけての心理学・コミュニケーション理論・キャリア理論を、20の学派（school）に分類し、45名の理論家とその影響関係を可視化しています。

これらは大きく**7つの独立した系譜**に集約できます：

| # | 系譜 | 起点 | 中核的問い | 学派 |
|---|------|------|---------|------|
| ① | 精神分析 | フロイト(1895) | 無意識は人間をどう動かすか | psychoanalysis, ta |
| ② | 行動療法→CBT | パブロフ(1890) | 行動と認知をどう変えるか | behaviorism, assertive, cognitive, cbt, thirdwave, mindfulness |
| ③ | 人間性心理学 | ロジャーズ(1940)/マズロー(1954) | 人間の潜在力をどう引き出すか | humanistic, nvc |
| ④ | 社会構成主義 | ガーゲン(1985) | 現実は言語でどう構成されるか | constructionism, narrative |
| ⑤ | キャリア理論 | パーソンズ(1908) | 職業選択と発達をどう支援するか | vocational, career_dev, decision, lifespan, happenstance, career |
| ⑥ | 社会的学習理論 | バンデューラ(1971) | 観察と信念は行動をどう変えるか | social_learning |
| ⑦ | 組織心理学 | レヴィン(1939) | 組織と個人の関係をどう最適化するか | org_career |

**重要**: これらは完全に独立しているのではなく、複数の交差点で繋がっています（第9章参照）。

## 2. 系譜① 精神分析とその派生

### 流れ

```
フロイト(1895) ──→ ユング(1930) ──→ エリクソン(1950)
                                        ↓
                                     バーン(1957, TA)
```

### 理論家と接続

**フロイト (Sigmund Freud, 1895, 没1939)**
精神分析の創始者。無意識・自我・イド・超自我の構造モデル。年表全体における最も多くの影響線の起点であり、以下の理論家に直接影響を与えた：
- ユング（弟子→決別）
- エリクソン（発達理論へ拡張）
- エリス（精神分析から出発→REBT）
- ベック（精神分析から出発→認知療法）
- マズロー（精神分析を踏まえつつ超克）

**ユング (Carl G. Jung, 1930, 没1961)**
フロイトの弟子だが1912-13年に決別。リビドーを性的エネルギーに限定せず普遍的心的エネルギーに拡張。「人生の諸段階」（1930-31）で人生前半（外的適応）と後半（内面・個性化）の二分を提唱。集合的無意識・元型・個性化。
- **一次資料検証**: NZZ 1930年3月初出、改訂版1931年（CW第8巻の脚注で確認）
- **後続への影響**: エリクソンのライフサイクル論、スーパーのキャリア発達段階、グローヴのシンボル理論

**エリクソン (Erik Erikson, 1950, 没1994)**
フロイト・ユング双方から影響。人生全体にわたる8段階の心理社会的発達モデルを構築。
- **後続への影響**: バーン（直接の師）、シュロスバーグ（トランジション理論）

**バーン (Eric Berne, 1957, 没1970)**
交流分析（TA）の創始者。精神分析から出発しエリクソンに師事。「親・大人・子ども」の自我状態モデル。
- **一次資料検証**: 1957年論文、1961年著書、1964年「Games People Play」

## 3. 系譜② 行動療法→認知療法→CBT第三波

### 流れ

```
パブロフ(1890) → サルター(1949) → ウォルピ(1958) → アルベルティ(1970)
                      ↓                    ↓
                   エリス(1955)         ラザラス(1976)
                      ↓
                   ベック(1963) → リネハン(1993, DBT)
                      ↓            ヘイズ(1999, ACT)
                      ↓
             シーガル他(1995, MBCT) ← カバット＝ジン(1979, MBSR)
```

### CBTの三つの波

この系譜は歴史的に「三つの波」として整理されます：

| 波 | 時期 | 焦点 | 代表的理論家 |
|---|------|------|-----------|
| **第一波** | 1947-1975 | 行動の変容（外から観察可能な行動） | パブロフ、サルター、ウォルピ |
| **第二波** | 1953-1988 | 認知の変容（思考の内容を変える） | エリス、ベック、マイケンバウム |
| **第三波** | 1977-2010 | 認知との関係性の変容（思考を変えるのではなく、思考への態度を変える） | リネハン(DBT)、ヘイズ(ACT)、シーガル他(MBCT) |

### 理論家と接続

**エリス (Albert Ellis, 1955, 没2007)**
REBT（合理情動行動療法）の創始者。精神分析から出発し、行動主義と人間性心理学の間の「第三の道」を切り開いた。ABCモデル（Activating event → Belief → Consequence）。
- **一次資料検証**: 本人の著書『Reason and Emotion in Psychotherapy』で「1955年初頭に基本理論が確立」と明記
- **影響源**: サルター（行動療法の先駆者として自ら認めた）、フロイト（精神分析訓練を経験）

**ベック (Aaron T. Beck, 1963, 没2021)**
認知療法/CBTの創始者。精神分析出身だが「自動思考・認知の歪み」に着目。うつ・不安への最大のエビデンス基盤を持つ療法を確立。
- **後続への影響**: リネハン(DBT)、ヘイズ(ACT)、シーガル他(MBCT) — CBT第三波すべてがベックの認知療法を出発点としている

**第三波の特徴的転換**:
第二波が「歪んだ思考の内容を修正する」ことを目指したのに対し、第三波は「思考との関係性を変える」方向に転換した。これはロジャーズ的な受容の姿勢やマインドフルネス（仏教瞑想の世俗化）との合流でもある。

## 4. 系譜③ 人間性心理学 — カウンセリング・コーチング・コミュニケーション

### 流れ

この系譜は年表の中で最も多くの理論家を含み、**2つの平行する流れ**があります：

```
【ロジャーズ系 — カウンセリング・コミュニケーション】
ロジャーズ(1940) → ゴードン(1962, PET/I-message)
                 → アイビィ(1968, マイクロカウンセリング)
                 → ミラー/ロルニック(1983, 動機づけ面接)
                 → グローヴ(1989, クリーン・ランゲージ) ← ユング(シンボル理論)
                 → ローゼンバーグ(1963, NVC) ← ゴードン
                 → グリーンリーフ(1970, サーバントリーダーシップ)

【マズロー系 — コーチング】
マズロー(1954) → ガルウェイ(1974, インナーゲーム)
              → ウィットモア(1992, GROWモデル) ← ガルウェイ
              → マクレガー(1960, Theory X/Y) ← レヴィン
```

### 重要な構造的理解

**カウンセリングとコーチングは「親子関係」ではなく「兄弟関係」**です。

共通の祖先は**人間性心理学（第三の力）**ですが、そこから2つの異なる流れが分岐しました：
- **ロジャーズ系**: 傾聴・共感・非指示的アプローチ → カウンセリング・MI・NVC
- **マズロー系**: 自己実現・潜在能力 → インナーゲーム → ビジネスコーチング

### 理論家と接続

**ロジャーズ (Carl Rogers, 1940, 没1987)**
クライアント中心療法の創始者。共感・無条件の肯定的関心・一致性の3条件。マズローと並ぶ人間性心理学（第三の力）の双頭。
- **後続への影響**: 年表の中で最も多くの後続理論家に影響を与えた人物の一人。ゴードン、アイビィ、ミラー/ロルニック、ローゼンバーグ、グローヴ、グリーンリーフの6名が直接の影響を受けた
- **一次資料検証**: ミラー/ロルニックの著書の謝辞で「Carl Ransom Rogers (1902-1987)への恩義を最初に認める」と明記

**マズロー (Abraham Maslow, 1954, 没1970)**
欲求階層説と自己実現理論。フロイト・アドラー・ユング・ホーナイ・ゴールドシュタインの「部分的真実を統合」（本人の言葉）。行動主義（第一の力）と精神分析（第二の力）を超える「第三の力」＝人間性心理学を確立。
- **一次資料検証**: 動機理論は1942年に精神分析学会で初発表、1954年著書『動機と人格』
- **後続への影響**: ロー（欲求階層→職業選択）、ガルウェイ（自己実現→インナーゲーム）、ウィットモア（欲求階層→コーチング理論基盤）、マクレガー（欲求階層→Theory X/Y）

**ガルウェイ (W. Timothy Gallwey, 1974)**
インナーゲームの創始者。高等教育を離れテニス指導の中で発見した原理。「頭の中の対戦相手こそ最強の敵」。学習者自身の自然な学習能力への深い信頼が基盤。
- **重要な位置**: カウンセラーではなくスポーツコーチ出身。コーチングがカウンセリングから派生したのではないことを示す重要な証拠
- **一次資料検証**: AT&T・Apple・IBMで20年以上ビジネスに適用（『The Inner Game of Work』2000年）

**ウィットモア (Sir John Whitmore, 1992, 没2017)**
GROWモデル（Goal・Reality・Options・Will）の体系化者。ガルウェイに師事しインナーゲームを英国に導入。
- **一次資料検証**: 『Coaching for Performance』の謝辞でガルウェイを「must undoubtedly head the list」と明記。本文でマズローの欲求階層を理論基盤として直接引用

**ミラー/ロルニック (William R. Miller / Stephen Rollnick, 1983)**
動機づけ面接（MI）の創始者。ロジャーズのパーソン・センタード・アプローチを基盤に、変化への動機を引き出す技法を体系化。
- **一次資料検証**: 著書4版の謝辞で「MIはロジャーズとその弟子たちの研究の堅固な基盤の上に構築されたパーソン・センタード・メソッド」と明記

**グローヴ (David Grove, 1989, 没2008)**
クリーン・ランゲージの開発者。Lawley & Tompkinsがシンボリック・モデリングとして体系化（2000年）。ロジャーズの非指示的アプローチの徹底的純化 — クライアントのメタファーをそのまま用い、セラピストの解釈を一切入れない。
- **影響源**: ロジャーズ（非指示的）+ ユング（シンボル理論）+ 認知言語学（Lakoff & Johnson）

## 5. 系譜④ 社会構成主義→ナラティブ

### 流れ

```
[哲学: Wittgenstein, Kuhn, Berger & Luckmann]
        ↓
ガーゲン(1985) → ホワイト/エプストン(1990, ナラティブ・セラピー) ← フーコー(権力論)
             → サビカス(2002, キャリア構成理論) ← スーパー
```

### 理論家と接続

**ガーゲン (Kenneth Gergen, 1985)**
社会構成主義の提唱者。「現実は言語と関係によって共同構成される」。
- **一次資料検証**: American Psychologist, Vol.40, No.3, March 1985
- **影響源**: Wittgenstein（言語哲学）、Kuhn（パラダイム論）、Berger & Luckmann（知識社会学）— いずれも哲学・社会学であり、心理学内部の系譜ではない。これがガーゲンが年表上で「孤立して見える」理由
- **重要な批判**: ローゼンバーグ(NVC)の「普遍的ニーズ」の前提を否定。「ニーズは個人の中に存在するのではなく、社会的に構成される」

**ホワイト/エプストン (Michael White / David Epston, 1990)**
ナラティブ・セラピーの創始者。問題の外在化「問題と人を分離する」。
- **一次資料検証**: Copyright 1990, Dulwich Centre。序文でフーコーの哲学的分析からの引用を明示

## 6. 系譜⑤ キャリア理論（特性因子・発達・意思決定）

### 流れ

```
パーソンズ(1908) → ロー(1956) ← マズロー(欲求階層)
                → ホランド(1959)
                → スーパー(1957) ← ギンズバーグ(1951), ユング(人生段階)
                      ↓
                   ハンセン(1997)
                   サビカス(2002) ← ガーゲン, ホワイト

エリクソン → シュロスバーグ(1981, トランジション)

ジェラット(1962, 意思決定)
```

### 理論家と接続

**パーソンズ (Frank Parsons, 1908, 没1908)**
職業指導の創始者。「自己分析・職業理解・合理的推論」の3ステップ。特性因子論の源流。
- 注: 年表上のyearは1908年（職業局設立・没年）。著書は死後の1909年出版

**スーパー (Donald Super, 1957, 没1994)**
キャリア発達理論。キャリアを自己概念の実現過程として定義。5段階モデル（成長→探索→確立→維持→解放）。
- **影響源**: パーソンズ（特性因子論の拡張）、ギンズバーグ（発達的アプローチの先駆）、ユング（人生段階論）
- **一次資料検証**: 1957年『The Psychology of Careers』、1984年ライフ・キャリア・レインボー

**サビカス (Mark Savickas, 2002)**
キャリア構成理論（CCT）。スーパーの理論を社会構成主義・ナラティブ心理学と統合。
- **一次資料検証**: CCT初出は**2002年**（従来の記載2005年は誤り。Career_Construction_Theory_Book_Savickas.pdfで確認）
- **影響源**: スーパー（キャリア発達）+ ガーゲン（社会構成主義）+ ホワイト（ナラティブ）— 3つの系譜の合流点

## 7. 系譜⑥ 社会的学習理論

### 流れ

```
バンデューラ(1971) → クランボルツ(1996, 偶発性理論)
                   → レント他(1994, SCCT)
                   → ドゥエック(1988, マインドセット)
```

### 理論家と接続

**バンデューラ (Albert Bandura, 1971, 没2021)**
社会的学習理論・自己効力感の提唱者。観察学習・モデリング・自己調整。
- **後続への影響**: クランボルツ（キャリア意思決定にバンデューラの社会的学習理論を応用）、レント他（自己効力感をキャリア領域に応用）、ドゥエック（社会認知理論の系譜）
- 注: クランボルツの影響源は当初「ウォルピ」と記載されていたが、一次資料検証で**バンデューラ**に修正

**ドゥエック (Carol Dweck, 1988)**
マインドセット理論。固定マインドセット vs 成長マインドセット。
- **一次資料検証**: 学術的基盤はDweck & Leggett (1988)論文。一般書『Mindset』は2006年

## 8. 系譜⑦ 組織心理学・リーダーシップ

### 流れ

```
レヴィン(1939, MIT) → マクレガー(1960, MIT Sloan) → シャイン(1978) → ホール(1976) → アーサー(1996)
                          ↑
                     マズロー(1954)

ロジャーズ(1940) → グリーンリーフ(1970, サーバントリーダーシップ)
```

### MIT系譜の重要性

この系譜はMIT（マサチューセッツ工科大学）を拠点とした組織心理学の知的系譜です：

1. **レヴィン (Kurt Lewin, 1939, 没1947)** — 1945年MITにグループ・ダイナミクス研究センターを設立。リーダーシップ3類型（専制・民主・放任）、変革の3段階モデル（解凍→変化→再凍結）
2. **マクレガー (Douglas McGregor, 1960, 没1964)** — MIT Sloan。レヴィンの組織研究基盤 + マズローの欲求階層 → Theory X/Y。Y理論はコーチング・サーバントリーダーシップの理論的先駆
3. **シャイン (Edgar Schein, 1978)** — MIT Sloan。マクレガーの後継者。キャリア・アンカー理論、組織文化論
4. **ホール (Douglas Hall, 1976)** — シャインに師事。プロティアン・キャリア理論
5. **アーサー (Michael Arthur, 1996)** — バウンダリーレス・キャリア

**グリーンリーフ (Robert Greenleaf, 1970, 没1990)**
サーバント・リーダーシップの提唱者。AT&Tで38年間勤務後に発表。「リーダーはまず奉仕者であれ」。傾聴・共感・癒し・気づきなど10特性。
- ロジャーズ的な傾聴・共感・人間の成長への信頼が、リーダーシップ論に昇華された形

## 9. 系譜間の交差点 — 理論の合流地点

年表の中で、複数の系譜が交差する重要なノードがあります：

### 交差点①: フロイト — 精神分析から3方向への分岐

```
フロイト → ユング → エリクソン（精神分析の内部発展）
       → エリス → ベック（精神分析への反発→認知療法）
       → マズロー（精神分析を踏まえつつ超克→人間性心理学）
```

### 交差点②: マズロー — 人間性心理学から4方向への展開

```
マズロー → ガルウェイ → ウィットモア（コーチング）
        → マクレガー → シャイン（組織心理学）
        → ロー（キャリア理論）
        → [ロジャーズと共に]人間性心理学（第三の力）
```

### 交差点③: サビカス — 3系譜の合流

```
スーパー（キャリア発達）  ──→
ガーゲン（社会構成主義）  ──→ サビカス（キャリア構成理論）
ホワイト（ナラティブ）   ──→
```

### 交差点④: グローヴ — 2系譜の統合

```
ロジャーズ（非指示的アプローチ） ──→
ユング（シンボル理論）          ──→ グローヴ（クリーン・ランゲージ）
```

## 11. よくある誤解と修正

### 誤解1: 「ロジャーズが心理学全体のベース」

**修正**: ロジャーズは人間性心理学系（カウンセリング・MI・NVC・コーチングの一部）の基盤であり、心理学全体のベースではない。行動療法・CBT・キャリア理論の多くはロジャーズとは独立に発展した。

### 誤解2: 「カウンセリングから派生してコーチングが生まれた」

**修正**: カウンセリング（ロジャーズ系）とコーチング（マズロー→ガルウェイ→ウィットモア系）は兄弟関係であり、親子関係ではない。共通の祖先は人間性心理学（第三の力）。ガルウェイはカウンセラーではなくテニスコーチ出身。

### 誤解3: 「問題解決＝カウンセリング、潜在能力＝コーチング」

**修正**: マズローの自己実現やロジャーズの「十分に機能する人間」はカウンセリングの文脈で潜在能力にフォーカスしている。逆にコーチングも問題解決を扱う。この二分法は業界のマーケティング的区別であり、理論的根拠は薄い。

### 誤解4: 「クライアントの中から答えを引き出す＝共通効果」

**修正**: これはロジャーズ派/人間性心理学の立場であり、心理療法全体の共通原理ではない。CBTは指示的に介入する。ガーゲン（社会構成主義）は「答えは個人の中に存在するのではなく、言語と関係性によって共同構成される」と批判している。心理療法の共通要因研究が特定しているのは、治療同盟・期待/希望・クライアント要因である。
