# 系譜年表レポートを活用するためのAIプロンプト集

> REPORT.md の内容をAI（Claude、ChatGPT、Gemini等）に渡して、学習資料を生成するためのプロンプト例です。
> 使い方: REPORT.md の全文をコピーし、以下のプロンプトの `{REPORT}` 部分に貼り付けてください。

---

## 1. スライド資料の生成

### 1-A. 全体概要スライド（30分講義向け）

```
以下のレポートは、心理学・コミュニケーション理論・キャリア理論の系譜年表（45名の理論家、7つの大系譜）をまとめたものです。

{REPORT}

このレポートをもとに、30分の講義で使用するスライド資料を作成してください。

【要件】
- スライド枚数: 15〜20枚
- 対象者: キャリアコンサルタント養成講座の受講生（心理学の基礎知識はあるが、系譜の全体像は把握していない）
- 構成:
  1. 導入: なぜ系譜を理解する必要があるか（3枚）
  2. 7つの大系譜の全体マップ（2枚）
  3. 各系譜の核心を1〜2枚ずつ（7〜14枚）
  4. 系譜間の交差点 — 理論がどこで合流するか（2枚）
  5. まとめ: よくある誤解と正しい理解（2枚）
- 各スライドに「話すべきポイント」をスピーカーノートとして付記
- 図解・フローチャートを多用し、文字だけのスライドを避ける
- 出力形式: Markdown（Marp、Slidev、またはGoogle Slides変換用）
```

### 1-B. 特定系譜の深堀りスライド

```
以下のレポートから「系譜③ 人間性心理学」のセクションを深堀りするスライド資料を作成してください。

{REPORT}

【要件】
- スライド枚数: 10〜15枚
- 対象者: コーチング/カウンセリングの実務者（「ロジャーズは知っているがマズローとの関係が曖昧」レベル）
- 特に強調すべきポイント:
  - ロジャーズ系（カウンセリング）とマズロー系（コーチング）の「兄弟関係」
  - MIはロジャーズ系、GROWモデルはマズロー系という構造
  - 「カウンセリングからコーチングが派生した」という誤解の修正
- 各理論家について「一次資料で確認された事実」と「よくある誤解」を対比
- 出力形式: Markdown（スピーカーノート付き）
```

## 2. 音声解説スクリプトの生成

### 2-A. ポッドキャスト形式（1人語り、20分）

```
以下のレポートをもとに、20分のポッドキャスト風の音声解説スクリプトを作成してください。

{REPORT}

【要件】
- 形式: 1人語り（解説者が聴取者に語りかける）
- トーン: 専門的だが親しみやすい。「〜なんですよね」「ここが面白いところで」など口語調
- 構成:
  1. オープニング: 「今日は心理学の系譜年表を使って、45人の理論家がどう繋がっているかを20分で解説します」（1分）
  2. 全体像: 7つの大系譜を地図のように概観（3分）
  3. ストーリーで辿る系譜:
     - フロイトから始まる分岐（精神分析→3方向）（3分）
     - ロジャーズとマズローの「第三の力」— カウンセリングとコーチングの兄弟関係（5分）
     - MIT系譜 — レヴィン→マクレガー→シャイン（3分）
     - サビカスに合流する3つの川（2分）
  4. よくある誤解3つとその修正（2分）
  5. クロージング（1分）
- 時間目安を各セクションに記載
- 「ここで年表を見てほしいのですが」など、年表アプリ（psychology-timeline.vercel.app）への誘導を自然に入れる
```

### 2-B. 対話形式（2人、30分）

```
以下のレポートをもとに、2人の対話形式の音声解説スクリプトを作成してください。

{REPORT}

【要件】
- 形式: ホスト（キャリアコンサルタント、基礎知識あり）× ゲスト（心理学の系譜に詳しい専門家）
- 30分（約8,000〜10,000字）
- 構成:
  1. 導入: ホストが「理論家の名前は知っているけど、繋がりが分からない」という悩みを提示（2分）
  2. ゲストが全体像を描く — 「7つの川があって、それが何箇所かで合流するイメージ」（5分）
  3. ホストが素朴な疑問をぶつける:
     - 「コーチングってカウンセリングから生まれたんですよね？」→ ゲストが修正（5分）
     - 「ロジャーズが全部のベースですよね？」→ ゲストが範囲を限定（5分）
     - 「CBTと人間性心理学って全然違うもの？」→ ゲストが第三波での合流を説明（5分）
  4. MIT系譜の話 — 「シャインって突然出てくる感じがしたんですけど」→ レヴィン→マクレガーの繋がり（5分）
  5. まとめと年表アプリの紹介（3分）
- ホストの疑問は「よくある誤解」セクションの内容を自然に反映
```

## 3. 学習用クイズ/確認テストの生成

### 3-A. 理解度チェッククイズ（20問）

```
以下のレポートをもとに、理解度チェック用のクイズを20問作成してください。

{REPORT}

【要件】
- 形式: 4択問題（1問1答、正解と解説付き）
- 難易度分布: 基礎10問 / 応用7問 / 発展3問
- 出題カテゴリ:
  - 「誰が誰に影響を与えたか」（影響関係）: 5問
  - 「この理論は何年に発表されたか」（年代）: 3問
  - 「この理論の核心は何か」（内容理解）: 5問
  - 「この2つの理論の違いは何か」（比較）: 4問
  - 「よくある誤解はどれか」（誤解の修正）: 3問
- 各問に「なぜこの答えが正しいか」の解説を100字程度で付記
- 出力形式: Markdown
```

## 4. 関連性マップ/図解の生成

### 4-A. Mermaid図の生成

```
以下のレポートの「付録: 全理論家一覧」をもとに、Mermaid記法で影響関係の全体図を生成してください。

{REPORT}

【要件】
- Mermaid flowchart (graph TD) 形式
- ノードの色を学派ごとに分ける（style指定）
- 影響関係を矢印で表現
- 7つの大系譜ごとにsubgraphで囲む
- 系譜間を跨ぐ影響線を破線で表現
- 出力: そのままMermaid対応ツール（GitHub、Notion、Obsidian等）に貼り付けて使えるコード
```

### 4-B. 概念地図（手書き風の説明図）

```
以下のレポートをもとに、手書きホワイトボード風の概念地図を作成するための指示書を書いてください。

{REPORT}

【要件】
- A3用紙1枚に収まる構成
- 中央に「人間性心理学（第三の力）」を配置
- そこから放射状に各系譜を配置
- 各理論家を円で表し、影響関係を矢印で接続
- 色分け指示（マーカー色の指定）
- 「ここは太く描く」「ここは破線」などの描画指示
- 対象: セミナーでホワイトボードに描きながら説明する講師向け
```

## 5. 要約・翻訳

### 5-A. エグゼクティブサマリー（A4 1枚）

```
以下のレポートを、A4用紙1枚（約1,500字）に要約してください。

{REPORT}

【要件】
- 対象読者: キャリアコンサルタント試験の受験生
- 構成: 全体構造（300字）→ 最重要の3系譜（各300字）→ よくある誤解（300字）
- 試験で問われやすいポイントを太字で強調
- 年代と人名の正確性を最優先
```

### 5-B. 英語版の生成

```
Please translate the following report into English, maintaining academic precision for theorist names, theory names, and dates. Keep the structure and formatting intact.

{REPORT}

Additional requirements:
- Use standard English transliterations for Japanese theorist names
- Keep theory names in their established English forms (e.g., "Planned Happenstance", "Career Construction Theory")
- Add translator's notes where Japanese-specific context needs explanation
```

## 使い方のヒント

1. **REPORT.md は必ず全文を渡す** — 部分的に渡すと系譜間の接続が切れます
2. **対象者のレベルを明示する** — 同じ内容でも初学者向けと実務者向けでは深さが変わります
3. **出力形式を指定する** — Markdown、HTML、Mermaid等を明示すると後処理が楽です
4. **年表アプリへの誘導を入れる** — https://psychology-timeline.vercel.app で実際に操作すると理解が深まります
5. **一次資料の検証結果を活かす** — 「〜と言われているが、一次資料では〜」という構成は説得力があります
